A plugin to embed the DOP Player to play flv, mov, mp4, m4v, m4a, mp4v, 3gp, 3g2 video files that you add to your post.

== Description == 

**Dop Player** is intended for users that want an easy to use player to integrate in their blog.

**Dop Player** can be easily customized from an admin panel to match your template's colors.

You don't need to know coding to use DOP Player.

**DOP Player** supports the folowing video file formats:

* .flv
* .mov
* .mp4
* .m4v
* .m4a
* .mp4v
* .3gp
* .3g2 

== Installation ==

1. Upload the content of the "**dop-player.1.0.zip**" file to "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Using Dop Player ==

Add a video to your post from "Post" menu, "Add new", "Upload/Insert - Add video" . The DOP Player plugin will automatically insert the player so you can visualize the video file. 

== Editing Dop Player ==

You can change width, height and the colors of the players from "DOP Player" administration area in "Settings" menu in WordPress.

== Frequently Asked Questions ==

= Why should I use Dop Player? =

You should use **Dop Player** because:

1. You can easy resize customize the player to mach your template (no coding needed).
2. You can easily add a movie using only WordPress admin panel (no need to add codes to your template).  
3. Supports all video formats supported by Adobe Flash Player.

= What video file formats does the DOP Player support? =

DOP Player supports the formats supported by Adobe Flash Player: 

* .flv
* .mov
* .mp4
* .m4v
* .m4a
* .mp4v
* .3gp
* .3g2
